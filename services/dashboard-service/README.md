# 대시보드 서비스 (Dashboard Service)

**Dashboard Service**는 사용자가 시스템과 상호작용하고 투자를 모니터링하는 **웹 인터페이스**입니다. Next.js를 기반으로 구축되어 빠른 렌더링과 SEO 최적화를 지원합니다.

## 🛠 주요 기능 (Features)

### 1. AI 에이전트 채팅 (Chat Interface)
-   **실시간 대화**: WebSocket을 통해 Cognitive Service의 에이전트와 실시간으로 대화합니다.
-   **스트리밍 응답**: LLM의 생성 텍스트를 토큰 단위로 스트리밍하여 빠른 응답성을 제공합니다.
-   **마크다운 렌더링**: 에이전트가 생성한 표, 코드, 강조 구문을 깔끔하게 렌더링합니다.

### 2. 실시간 시장 모니터링
-   **호가창 (Orderbook)**: Kafka에서 수신한 실시간 호가 데이터를 시각화합니다.
-   **차트 (Chart)**: TradingView 라이브러리 등을 활용하여 캔들 차트와 기술적 지표를 표시합니다.

### 3. 포트폴리오 관리
-   **자산 현황**: 현재 보유 종목, 평가 금액, 수익률을 실시간으로 갱신합니다.
-   **주문 내역**: 체결된 주문과 미체결 주문(Pending) 목록을 보여줍니다.

### 4. 백테스트 리포트 뷰어
-   **결과 시각화**: Backtesting Service가 생성한 수익률 곡선과 통계 데이터를 그래프로 보여줍니다.

## 🚀 시작하기 (Getting Started)

### 의존성 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```
