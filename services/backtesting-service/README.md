# 백테스팅 서비스 (Backtesting Service)

**Backtesting Service**는 과거 데이터를 기반으로 전략의 수익성과 안정성을 검증하는 **시뮬레이션 엔진**입니다. 실제 매매 환경과 유사한 조건을 구현하여 전략의 과최적화를 방지합니다.

## 🛠 주요 기능 (Features)

### 1. 이벤트 기반 시뮬레이션 (Event-Driven Simulation)
-   **Tick-by-Tick 재생**: 과거 데이터를 한꺼번에 연산(Vectorized)하지 않고, 시간 순서대로 하나씩 이벤트를 발생시켜 처리합니다.
-   **Lookahead Bias 제거**: 전략은 오직 '현재 시점'까지의 데이터만 볼 수 있어, 미래 데이터를 참조하는 오류를 원천 차단합니다.

### 2. 현실적 체결 모델링
-   **지연 시간 (Latency)**: 주문 발생 후 거래소 도달까지의 시간을 시뮬레이션하여, 급변하는 시장에서의 체결 실패 가능성을 반영합니다.
-   **슬리피지 (Slippage)**: 호가창의 잔량을 고려하여, 요청한 가격보다 불리한 가격에 체결되는 현상을 구현합니다.
-   **수수료 및 세금**: 실제 거래 비용을 차감하여 순수익을 계산합니다.

### 3. 성과 분석 리포트
-   **주요 지표 산출**: CAGR(연평균 수익률), MDD(최대 낙폭), Sharpe Ratio(샤프 지수) 등을 계산합니다.
-   **시각화 데이터 생성**: 자산 곡선(Equity Curve) 및 월별 수익률 히트맵 데이터를 생성하여 대시보드로 전송합니다.
